## 3. v2.0 详细需求

### 3.1 P0-1 移动端优化

**需求描述：**
优化移动端（< 768px）的交互体验

**验收标准：**
- [ ] 首页卡片在移动端单列显示
- [ ] 底部导航栏（移动端）
- [ ] 滑动手势切换视图
- [ ] 图片预览支持缩放
- [ ] 弹窗全屏显示（移动端）

### 3.2 P0-2 云同步基础

**需求描述：**
集成 Supabase 实现跨设备数据同步

**技术方案：**
- 使用 Supabase 作为 BaaS
- 本地优先，后台同步
- 冲突解决：最后写入胜出

**验收标准：**
- [ ] 用户可登录/注册
- [ ] 日记自动同步到云端
- [ ] 换设备登录后数据恢复
- [ ] 离线时本地存储，联网后同步

### 3.3 P0-3 日记编辑增强

**需求描述：**
支持 Markdown 格式和富文本编辑

**验收标准：**
- [ ] 支持 Markdown 语法
- [ ] 实时预览
- [ ] 工具栏快捷操作
- [ ] 代码块语法高亮
